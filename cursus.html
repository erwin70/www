Maak een prachtige website

Ben jij klaar om jouw ideeën tot leven te brengen? Of u nu een bedrijf wilt starten, uw vaardigheden wilt laten zien of uw creatieve projecten met de wereld wilt delen, onze cursus heeft alles wat u nodig heeft om een prachtige webpagina te maken.

Onze ervaren instructeurs begeleiden u bij het ontwerpen en coderen van een professionele website. Je leert de ins en outs van HTML, CSS, JavaScript en Git, waardoor je de vaardigheden krijgt om op te vallen in het huidige competitieve digitale landschap.

Met onze cursus leer je hoe je jouw verhaal online op een boeiende manier kunt vertellen. Van het kiezen van het perfecte kleurenschema tot het optimaliseren van uw website voor zoekmachines, wij leren u de strategieën en technieken om uw webpagina te laten schitteren.

Wat wij doen

Wij bieden op dit moment 5 verschillende cursussen aan zodat je zelf aan de slag kan met jouw website
of maken jouw website op maat
of we doen het samen.

Git en GitHub:

De beste manier om aan jouw website te werken,
met Git werk je lokaal op jouw computer aan je website en
GitHub is een ontwikkelplatform waar je jouw code kan hosten
en met meerdere tegelijk aan jouw site kan werken.
Alle stappen die je neemt worden opgeslagen en
je kan altijd terug gaan naar eerdere creaties.

HTML 5:

De basis om jouw website te maken.

CSS:

Cascading Style Sheets, is een programmeertaal
die zorgt voor de opmaak van HTML bestanden.
CSS zorgt ervoor dat als je graag een verandering brengt
in het uiterlijk van je site je niet in elke pagina
alles moet aanpassen maar simpelweg in je CSS file.

Javascript:

Javascript wordt gebruikt om webpaginas interactiever en meer levend te maken.
Deze programmeertaal geeft je vele mogelijkheden die je niet kan uitvoeren met HTML of CSS.

SQL:

Alles om een database te koppelen aan je website.
